# ✊📄✂️ Stone Paper Scissors Game

This project is a web-based implementation of the classic Stone Paper Scissors (Rock Paper Scissors) game developed using HTML, CSS, and JavaScript. It allows users to play against the computer in an interactive and engaging environment with real-time results and score tracking.

The project demonstrates core front-end development concepts such as DOM manipulation, event handling, and logical decision-making using JavaScript.

---

## 🚀 Features

- 🎮 User vs Computer gameplay  
- ✊📄✂️ Three selectable options: Stone, Paper, Scissors  
- 🤖 Random computer choice generation  
- 🏆 Instant result display (Win / Lose / Draw)  
- 📊 Score tracking system  
- 🔄 Play again / reset functionality  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🎨 Clean and intuitive user interface  

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript (ES6)** – Game logic, randomization, and DOM manipulation

## ⚙️ How It Works

- The user selects one of the three options: Stone, Paper, or Scissors.  
- The computer generates a random choice.  
- The game compares both selections based on predefined rules:
  - Stone beats Scissors  
  - Paper beats Stone  
  - Scissors beats Paper  
- The result is displayed instantly as Win, Lose, or Draw.  
- The score is updated after each round.  
- Users can restart the game anytime using the reset option.
